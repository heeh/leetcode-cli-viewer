* Leetcode Dialog

** Dependency
*** 1. GNU Dialog
Linux
#+begin_src org :eval never-export
sudo apt install dialog
#+end_src
**** Mac
#+begin_src org :eval never-export
brew install dialog
#+end_src

*** 2. leetcode-cli
#+begin_src org :eval never-export
# https://github.com/clearloop/leetcode-cli
cargo install leetcode-cli
#+end_src

** Run
#+begin_src org :eval never-export
$ ./leetcode-dialog.sh
#+end_src

** Customize Theme
#+begin_src org :eval never-export
dialog --create-rc ~/.dialogrc
#+end_src
